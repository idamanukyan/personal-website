Selected Work

Government E-Tax Platform

Tax filing infrastructure for three nations.

The Challenge

Kosovo, Zimbabwe, and the Federated States of Micronesia needed secure, reliable e-tax systems capable of handling millions of citizens — with strict compliance requirements, limited existing infrastructure, and zero tolerance for downtime during filing periods.

The Approach

Designed a modular microservices architecture using Java and Spring Boot that could be adapted per country while sharing core tax calculation logic. Implemented OAuth2 with role-based access control to handle sensitive citizen data. Built high-throughput transaction processing with PostgreSQL, optimizing for peak filing periods. Prioritized audit logging and data integrity over feature velocity.

The Outcome

Deployed across 3 countries. Processing millions of tax filings annually. Zero security incidents. System became the foundation for broader e-governance initiatives in each nation.

Java Spring Boot Hibernate PostgreSQL OAuth2 REST APIs

Deutsche Leasing Document Pipeline

Automated conversion at enterprise scale.

The Challenge

Deutsche Leasing was manually processing thousands of legacy .doc and .docx files monthly — a slow, error-prone workflow that consumed significant staff hours and delayed downstream operations.

The Approach

Architected an automated document conversion pipeline with intelligent error handling and retry logic. Built monitoring dashboards to track processing status in real-time. Designed for fault tolerance — partial failures wouldn't block the entire batch. Implemented comprehensive logging for audit compliance.

The Outcome

10,000+ files processed monthly. 99.8% success rate. 85% reduction in manual processing time. Staff reallocated to higher-value work.

Java Spring Boot Python REST APIs PostgreSQL

HyperFit AI Commerce Platform

ML-driven product intelligence for fitness e-commerce.

The Challenge

A fitness e-commerce platform with 500+ products struggled with poor product discovery. Users couldn't find relevant gear and supplements. Search was keyword-based and missed semantic intent. No personalization meant missed conversion opportunities.

The Approach

Built an ML-driven recommendation engine that analyzed user workout patterns to suggest relevant products. Implemented NLP-based semantic search that understood intent, not just keywords. Designed the system to learn from user behavior over time. Kept the architecture simple enough for a small team to maintain.

The Outcome

Semantic search across 500+ products. Personalized recommendations based on workout history. Measurable improvement in product discovery and user engagement.

Java Spring Boot Python NLP Machine Learning PostgreSQL
