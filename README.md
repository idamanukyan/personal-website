# Personal Portfolio Website

A modern, bilingual (English/German) portfolio website for Ida Manukyan — Software Engineer based in Berlin.

## Features

- **Bilingual Support** — Toggle between English and German
- **Responsive Design** — Mobile-first, works on all devices
- **Contact Form** — Integrated with Formspree for email delivery
- **Modern Aesthetic** — Warm gold accent, dark theme, smooth animations
- **SEO Optimized** — Meta tags, Open Graph, JSON-LD schema

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- [Formspree](https://formspree.io) for form handling
- [Google Fonts](https://fonts.google.com) (Space Grotesk, Inter)

## Sections

- Hero with availability status
- About
- Selected Work (Case Studies)
- Experience
- Testimonials
- Contact Form

## Local Development

Open `index.html` in your browser, or run a local server:

```bash
python3 -m http.server 5500
```

Then visit `http://localhost:5500`
